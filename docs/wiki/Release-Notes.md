# Release Notes

## 1.0.44 / 45

This is the final local Android build currently published in the repository.

Highlights:

- Full-screen Android WebView packaging
- Offline bundled game runtime
- 21-stage platform game progression
- Save/load unlock from Level 6
- Level select unlock from Level 10
- Randomized treasure rewards
- Power spring effect lasting 13 seconds
- Rage potion effect lasting 10 seconds
- Weather and illumination systems
- Natural rhythm option
- Moon phase rendering for night scenes
- Multi-language UI and pickup prompts
- Mobile touch controls with combo input handling
- Larger HUD and phone-oriented control layout
- Refined start overlay, settings panel, and gameplay hints
- Runtime QA checkpoints for layout, input, localization, progression, effects, controls, and levels
